# Design

Shared design resources for the iMadeFire agent skills library. All HTML report outputs and skill-generated UIs should follow these guidelines to ensure a consistent, professional look across every skill.

## Design Systems

| System | Description |
|--------|-------------|
| [iMadeFire-ui/](iMadeFire-ui/) | The primary design system — dark FinTech dashboard aesthetic used across all report outputs |

## How to Use

When building a new HTML report skill, reference the relevant design system's README before writing any CSS. Copy the CSS variable block verbatim and follow the component patterns documented there.
