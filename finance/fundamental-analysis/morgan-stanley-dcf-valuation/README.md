# Morgan Stanley DCF Valuation

**Type:** Entry Point
**Author:** Yury Primakov
**Category:** Fundamental Analysis
**Companions:** 2 (MD Report, HTML Report)

## Overview

A Morgan Stanley-style Discounted Cash Flow valuation skill that takes a ticker symbol and produces a full 9-step DCF model, including revenue projections, WACC calculation, terminal value (both methods), sensitivity analysis, and a final investment verdict.

## How to Use

Invoke this skill when you want a DCF valuation for a specific stock.

**Trigger phrases:**
- "Run a DCF on [TICKER]"
- "What's the intrinsic value of [COMPANY]?"
- "Build a DCF model for [TICKER]"
- "Is [STOCK] undervalued?"

## What It Does

1. **Validates input** — asks for ticker if not provided
2. **Builds the 9-step valuation framework:**

| Step | Output |
|------|--------|
| Revenue Projections | 5-year model with growth assumptions |
| Margin Estimates | EBIT margin trajectory by year |
| Free Cash Flow | FCF calculation with all line items shown |
| WACC | CAPM-based cost of equity + after-tax cost of debt |
| Terminal Value | Exit multiple method + perpetuity growth method |
| Sensitivity Analysis | 5×5 grid of fair value vs. WACC and terminal growth |
| Price Comparison | Intrinsic value vs. current market price |
| Final Verdict | Undervalued / Fairly Valued / Overvalued + thesis |
| Risk Assessment | Top 5 model-breaking assumptions ranked by impact |

3. **Generates two output files** via companion skills

## Inputs

| Input | How Collected | Required |
|-------|--------------|----------|
| Ticker symbol | User message | Yes |
| Company name | User message or inferred from ticker | Yes |

## Outputs

|--------|-------------|--------|
