# Goldman Sachs Stock Screener

**Type:** Entry Point
**Category:** Fundamental Analysis
**Companions:** 2 (MD Report, HTML Report)

## Overview

A Goldman Sachs-style stock screening skill that guides the user through a structured investment profile questionnaire, screens 10 stocks tailored to their profile, and produces both a Markdown and an interactive HTML report.

## How to Use

Invoke this skill when you want to screen stocks based on your investment goals.

**Trigger phrases:**
- "Screen stocks for me"
- "Find the top 10 stocks for my profile"
- "Run the Goldman Sachs stock screener"

## What It Does

1. **Collects your investment profile** via a one-question-at-a-time multiple choice questionnaire:
   - Risk tolerance
   - Investment amount
   - Time horizon
   - Preferred sectors
   - Every question includes a custom free-text option (Z)

2. **Confirms your profile** in a summary table before proceeding

3. **Screens 10 stocks** across your selected sectors using a 9-step framework:
   - Valuation (P/E vs. sector avg)
   - Revenue growth (5-year trend)
   - Financial health (D/E ratio)
   - Income (dividend yield + payout sustainability)
   - Competitive moat (weak / moderate / strong)
   - 12-month bull & bear price targets
   - Risk rating (1–10 with reasoning)
   - Entry zones and stop-loss levels

4. **Generates two output files** via companion skills:
   - `gs-stock-screener-report-[DATE].md` — structured Markdown report
   - `gs-stock-screener-report-[DATE].html` — interactive FinTech dashboard

## Inputs

| Input | How Collected | Required |
|-------|--------------|----------|
| Risk tolerance | Multiple choice (Q1) | Yes |
| Investment amount | Multiple choice (Q2) | Yes |
| Time horizon | Multiple choice (Q3) | Yes |
| Preferred sectors | Multiple choice, multi-select (Q4) | Yes |

## Outputs

| Output | Generated By | Format |
|--------|-------------|--------|
| Markdown report | `stock-screener-md-report` companion | `.md` |
| HTML dashboard | `stock-screener-html-report` companion | `.html` |

## Companions

| Skill | Purpose |
|-------|---------|
